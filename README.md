# How to start
1. Clone repository
```shell
git clone https://github.com/EymireWorld/weather-api.git
```
2. Export requirements
```shell
poetry export --output requirements.txt --without-hashes
```
3. Start docker container
```shell
docker compose up -d
```
4. Enjoy)
